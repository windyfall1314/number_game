# number_game
数织游戏
数织（也被称为像素谜题或 Picross）的目标是根据棋盘边缘的数字提示，还原出隐藏在网格背后的像素图案。

1. 核心规则

棋盘的上方（列提示）和左侧（行提示）都有数字。这些数字代表了该行或该列中连续填充方块的组数和每组的数量。

单个数字：例如提示是 5，表示该行/列有连续 5 个填色格子。

多个数字：例如提示是 3 1，表示该行/列有两组格子。第一组是连续 3 个，第二组是 1 个。

间隔规则：关键点！ 不同的数字组之间，必须至少有一个空格（或打叉标记）。

2. 操作技巧

在我们的版本中，提供了两种工具：

填色笔 (Pen)：用于标记你确定需要填充的格子。

标记叉 (X)：用于标记你确定一定是空白的格子。这能帮你排除干扰。

电脑端建议：左键填色，右键打叉。

3. 解题思路（从易到难）

第一步：找“满行”或“大数字”

如果一个 5x5 的棋盘提示是 5，那么这一整行直接填满。
如果是 4，虽然不能直接填满，但中间的 3 个格子无论如何都会被填到（你可以先填中间，再推导两端）。

第二步：寻找“零”

如果提示是 0，说明整行/整列都是空的。果断把整行都打上“X”。

第三步：利用已填好的格子

一旦你填好了一部分，观察边缘数字。如果提示是 1 且你已经填了一个格子，那么这一行剩下的地方都可以打“X”。

第四步：排除法与逻辑推导

如果提示是 1 1 1，而你把三个格子连在一起填了（变成了一个 3），这就不符合逻辑了。必须保证它们之间有空位。

4. 常见误区提醒

不要瞎猜：数织是纯逻辑游戏，每一步都可以推导出来。

顺序固定：提示数字的顺序就是格子出现的顺序。例如 2 1 表示先出现 2 个连续格子，再出现 1 个。

纠错模式：如果你卡住了，点击游戏下方的“纠错模式”，系统会帮你标记出那些“不该填色却被你填了”的格子（显示红色边框）。
